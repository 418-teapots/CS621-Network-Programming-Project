

# Zlib Instruction

## 1. Install

wget http://zlib.net/zlib-1.2.11.tar.gz

tar zxf zlib-1.2.11.tar.gz

cd zlib-1.2.11

./configure

make

make check

sudo make install


## 2. Compile 

gcc zlib-test2.cc -lz -o zlib-test2


## 3. Run 

./zlib-test2
